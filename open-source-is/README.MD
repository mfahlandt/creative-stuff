## Open Source is

This Creative Commons design serves as a visual manifesto, designed to be freely used, adapted, and shared by all. It distills the movement into essential human-centric terms—ART, COMMUNITY, INCLUSION, DIVERSITY, FRIENDSHIP, FREEDOM, and TRUST—moving beyond mere software or legal licenses to emphasize the core human connections and shared values that make collaborative development possible.
The dynamic rainbow gradient signifies a dynamic, continuously flowing spectrum that represents the diverse perspectives and global unified effort behind open innovation. This design was unveiled for the first time at the Maintainer Summit in Amsterdam as a unifying symbol of appreciation for those who build and sustain our shared digital infrastructure.
